# HEARTBEAT.md — 主动智能检查

## 📅 日历检查（每次心跳）
工作日 09:00-18:00，查询未来 2h 日程，<30min 立即提醒+上下文，30-120min 简要提醒。格式：📅名称 ⏰时间 📍地点 📋上下文

## 📋 今日摘要（每天首次心跳）
读取今日/昨日日志 + 全天日程，生成摘要：日程+未完成+重点

## ⚡ 异常检测（低频）
memory-state.json 健康度 / Git 同步 / 最后对话间隔

## 🧠 DMN 空闲模式
触发：距上次 >30min + 无子会话。只读扫描 memory-state/skill-quality/今日日志。无异常→NO_REPLY，有异常→觉知报告。不整合/不修改/不调用工具。
