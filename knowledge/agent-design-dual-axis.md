## 纵轴: 认知功能七脉 (Resources)

```
外圈 ─ 治理 (信任) ─→ 「能上线」
  ↑
中圈 ─ 反思 (校正) · 协作 (分工) ─→ 「可靠」
  ↑
内圈 ─ 感知 (注意力) → 记忆 (连续性) → 推理 (不确定性) → 行动 (不可逆性) ─→ 「能用」
```

**三圈顺序不可逆**: 内圈跑顺 → 中圈可靠 → 外圈上线

## 横轴: 执行拓扑六式 (Error Flow)

| 拓扑 | 关键字 | 错误特性 |
|------|--------|---------|
| 链式 | 传 A→B→C | 顺流而下 |
| 路由 | 选 分类分发 | 入口即决 |
| 并行 | 合 多路 merge | 合龙临界 |
| 编排 | 拆 中心调度 | 边界即错 |
| 循环 | 修 生成-评估 | 停时决定 |
| 层级 | 管 父子分层 | 隔离或雪崩 |

核心洞察: **数据走哪条路，错误就走哪条路。**

## 使用口诀

1. 任务烧哪种资源? (纵轴)
2. 数据怎么流，错了往哪跑? (横轴)
3. 缺了哪一脉? 那条路上有人拦吗?

> 纵七脉定资源得失，横六式断错误去路。
