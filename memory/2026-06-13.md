### Wechatsync 安装指引

1. 打开 Chrome
2. 地址栏输入 `chrome://extensions/`
3. 右上角开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择文件夹：`C:\tools\wechatsync-extension`
6. 点击扩展图标 → 设置 → 开启「MCP 连接」
7. 复制 Token，运行：
```powershell
[System.Environment]::SetEnvironmentVariable("WECHATSYNC_TOKEN", "你的token", "User")
```
