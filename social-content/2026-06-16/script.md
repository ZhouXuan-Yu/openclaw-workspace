# 视频脚本 — 从聊天工具到工作单元

## 场景 1 / Hook (0-12s)
半年前，我还只是用 ChatGPT 聊天。
现在，我的 AI 助手能独立完成深度研究、分析 20 多个开源项目的系统设计、
还能自己发现 Bug 并修复。
这不是模型变强了，而是 Agent 的工程方式变了。

## 场景 2 / 工程化三层 (12-25s)
2026 年，AI Agent 已经进入工程化时代。
核心是三层架构：
最底层是 Loop 循环，负责感知、推理、行动、观察。
中间层是 Harness 运行底座，管理约束、反馈、安全和错误恢复。
最上层是 FDE，把 Agent 嵌入真实业务产生价值。

## 场景 3 / 实战验证 (25-38s)
我在自己的 OpenClaw 助手上做了验证。
从 v4 升级到 v5，引入了三大机制：
来自 Devin 的 Think Tool，关键决策前强制思考；
来自 Kiro 的 Task Mode Router，五种任务模式自动切换；
还有完成前自检清单，确保每一步都经过验证。
健康检查从 B 级升到 A 级。

## 场景 4 / 教育落地 (38-50s)
教育行业是 Agent 最佳落地场景之一。
对学生，AI 学伴可以个性化辅导，不给答案而是引导思考。
对教师，智能助教自动生成教案、批改作业、分析学情。
对学校，混合架构部署私有大脑，数据不出校，安全合规。

## 场景 5 / 结论 + CTA (50-60s)
Agent 的未来不在更强的模型，而在更稳的系统。
记忆、技能、安全、进化——这四件事做好了，Agent 才能从工具变成同事。
关注我，一起探索 AI Agent 的实战落地。
