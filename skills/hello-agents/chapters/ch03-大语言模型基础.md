# 第3章 大语言模型基础

## Core Idea
LLM 是现代智能体的"大脑"。理解从 N-gram → 神经网络语言模型 → Transformer 的演进，是掌握 Agent 构建的根基。

## Frameworks Introduced
- **N-gram 语言模型**: 基于马尔可夫假设的统计语言模型
- **词嵌入 (Word Embedding)**: 将离散符号映射到连续向量空间
- **Transformer (Vaswani et al. 2017)**: 完全基于自注意力的架构
- **位置编码 / 多头自注意力**: 为序列提供位置信息 + 并行捕获不同子空间注意力

## Key Concepts
- **马尔可夫假设** — 词的概率只与前面有限 n 个词有关
- **数据稀疏性 (Sparsity)** — N-gram 未出现序列概率为 0
- **余弦相似度** — 衡量词向量语义相似性
- **King - Man + Woman ≈ Queen** — 词向量的语义类比
- **长期依赖问题** — RNN 梯度消失无法捕获长距离依赖
- **LSTM 门控机制** — 遗忘门/输入门/输出门 + 细胞状态
- **编码器-解码器** — Transformer 宏观架构
- **自注意力 (Self-Attention)** — 每个 token 与所有 token 建立关联
- **因果语言建模** — 预测下一个 token 的预训练任务

## Mental Models
1. **向量空间语义**: 词的语义 ≈ 向量空间位置，数学运算 ≈ 语义运算
2. **注意力预算**: 每个 token 注意力有限，长上下文 token 间关联被"拉薄"
3. **并行 > 串行**: 自注意力让所有位置可并行计算

## Anti-patterns
- 词序列从未出现≠概率为 0，平滑仅缓解非根治
- 静态嵌入无法捕捉一词多义
- 忽略上下文腐蚀 — 输入越长，模型回忆信息能力下降

## Worked Example
**词向量类比**: King(0.9,0.8) - Man(0.7,0.9) + Woman(0.7,0.3) = (0.9,0.2) → 与 Queen(0.9,0.2) 余弦相似度 1.0，证明词向量捕获了性别和皇室抽象概念。

## Key Takeaways
1. LLM 的推理能力源于海量预训练，非手工编码
2. 自注意力是 Transformer 并行性和长程依赖捕获能力的根源
3. 词嵌入是语义在向量空间的几何映射
4. 当前 LLM 仍面临上下文窗口限制和事实幻觉
